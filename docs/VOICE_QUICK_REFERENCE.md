# Voice Quick Reference Card

## 🎙️ Voice Mode Activation

```
1. Open: http://localhost:4768/voice-strategy-center
2. Click: 🎙️ Voice Mode toggle
3. Ready: Push-to-talk interface appears
```

---

## 🎯 Push-to-Talk

### Button
```
Press & Hold 🎙️  →  Speak  →  Release
```

### Keyboard
```
Press SPACE  →  Speak  →  Release SPACE
```

---

## ⚙️ Voice Settings

Click **⚙️** to configure:

| Setting | Range | Default | Purpose |
|---------|-------|---------|---------|
| Voice | System voices | Auto | Choose speaker |
| Speed | 0.5x - 2.0x | 1.0x | Speech rate |
| Pitch | 0.5 - 2.0 | 1.0 | Voice tone |
| Auto-Speak | On/Off | On | Auto-play responses |

## 🏎️ Racing Tips

1. **Lower game audio** when asking questions
2. **Use straights** for complex queries
3. **Quick commands** during action
4. **Speed 1.0x-1.2x** for fast responses
5. **Auto-speak ON** for hands-free

---

## 🔧 Troubleshooting

**No mic permission?**
→ Check browser settings, grant access

**Voice not working?**
→ Use Chrome/Edge browser

**Can't hear responses?**
→ Check volume, enable auto-speak

**Voice cuts out?**
→ Hold PTT longer, speak clearly

✅ **Clear questions** - Be specific  
✅ **Pause first** - Wait for beep  
✅ **One question** - Per transmission  
✅ **Practice timing** - Gets easier!
