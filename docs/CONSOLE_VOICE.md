# Console (Xbox) audio vs PC engineer voice

**Implemented path:** the race engineer **voice and LLM** run on the **Windows PC** (headset / speakers on the PC). The **Xbox** only **sends game telemetry** over the network to Pits n' Giggles on that PC.

**Xbox "through the console" audio** (party chat, TV speakers only, wireless controller routing) is a different product path: you would need platform-specific capture, HDMI/audio routing, or a headset that pairs with **both** Xbox and PC—**not** included here.

Use the PC for engineer audio while sim racing; keep the **game on Xbox** and **telemetry to the same PC** as today.
