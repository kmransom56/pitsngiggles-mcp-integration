## 🚀 How to Deploy

### For End Users (Docker - Recommended)

```bash
# 1. Clone repository
git clone https://github.com/ashwin-nat/pits-n-giggles.git
cd pits-n-giggles

# 2. Start services
./start-mcp.sh    # Linux/macOS
start-mcp.bat     # Windows

# 3. Configure API key (optional but recommended)
# Edit .env.mcp and add:
# LLM_API_KEY=your_openrouter_or_openai_key

# 4. Restart if you added API key
./stop-mcp.sh && ./start-mcp.sh

# 5. Start F1 game with UDP telemetry enabled

# 6. Access Strategy Center
http://localhost/strategy-center          # AI chat
http://localhost/voice-strategy-center    # Voice mode
```

### For Developers (Traditional)

```bash
# 1. Clone and install Pits N Giggles
git clone https://github.com/ashwin-nat/pits-n-giggles.git
cd pits-n-giggles
./start.sh

# 2. Access
http://localhost:4768/strategy-center
```
